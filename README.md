Overview

This project explores the long-term evolution of U.S. motor vehicle sales, comparing automobile and light truck trends over several decades.
Created as part of a Data Storytelling course, the goal was to transform sales data into a clear narrative that communicates economic and consumer shifts through effective visualization and presentation design.

Objectives

Compare sales patterns between automobiles and light trucks.

Identify key turning points and external factors influencing vehicle sales.

Use visualization and storytelling to communicate insights to a general audience.
